# Fix Chat Animal And Station Event Messages

Type: AFK

## What

Animal messages and “on station” messages should render like normal chat messages with sender context and left/right alignment, not as anonymous icon rows.

Treat animal sightings and pass-position events as normal chat messages for layout: own events align right, other users align left with sender name, and both show timestamp. Keep a small supporting icon with the text, but do not render them as anonymous system rows.

For own event bubbles, do not show the sender name. For other users' event bubbles, show the sender name just like normal chat messages.

Use the normal chat bubble colors. Let only the small icon color hint at the event type.

Keep these messages automatically generated by the existing flows. This issue is display-only and should not change message storage or sending behavior.
